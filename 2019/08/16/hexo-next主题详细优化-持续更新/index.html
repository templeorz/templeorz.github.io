hexo 创建文章、标签、分类

0x01

  1. 创建文章

    1. 在hexo下创建一个新的文章

      `hexo new “文章名称”

      会直接显示路径一般都在主目录的/source/_posts下

  2. 文章基本设置

0x02

  1. 创建标签页面

    hexo new page tags

  2. 标签基本设置

    在/source/tag 下有个index.md ，更改里面的设置，要是不做的话，还是不行

0x03

  1. 创建分类页面

    hexo new page categories

  2. 分类基本设置，在/source/ categories下有个index.md

0x04

​ 如果想每次默认加上标签如categories，需在主目录的scaffolds文件夹下post.md修改
